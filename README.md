# 💧 Agua Inc. - SmartFlow Consumer App

Welcome to the central repository for the Agua Inc. SmartFlow mobile application and web portal. This application connects via Bluetooth to our flagship SmartFlow water bottles to track user hydration, calculate mineral intake, and manage subscription filter replacements.

## 🚀 Tech Stack
* **Frontend:** React Native (Mobile), Next.js (Web)
* **Backend:** Node.js / Express
* **Database:** PostgreSQL (User Data) / MongoDB (IoT Sensor Time-Series Data)

## 🛠 Branching Strategy
We enforce strict branch naming conventions for Jira/ClickUp tracking. 
All branches must follow: `feature/TASK-ID-description` or `bugfix/TASK-ID-description`
